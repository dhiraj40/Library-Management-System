# 📚 Library Management System

A dynamic web-based Library Management System built using **PHP, MySQL, HTML, CSS, and Vanilla JavaScript**, designed to automate library operations such as book inventory management, user handling, and borrowing workflows.

---

## 🚀 Features

### 👨‍💼 Admin Module
- Add, update, and delete books
- Manage student records
- Approve or reject book issue requests

### 👨‍🎓 Student Module
- Browse and search books
- Request book issuance
- View issued books and status

### 🔄 Book Lifecycle
- Approval-based issue system
- Book return handling

### 🔐 Authentication
- Secure login system for Admin and Students

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, Vanilla JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  

---

## 🧠 Engineering Highlights

- Built a fully dynamic web application **without using frontend frameworks**
- Implemented server-side rendering using PHP
- Managed client-side interactivity using Vanilla JavaScript
- Designed role-based access control (Admin vs Student)
- Developed an approval-based workflow for book issuance
- Modular structure with reusable PHP components
